# ♟️ שחמט - משחק שחמט Hebrew

משחק שחמט בעברית - שחקן נגד מחשב

## 🚀 תכונות

- 🎮 שחק מול מחשב עם 3 רמות קושי
- 📱 תאימות מלאה לנייד ולמחשב
- 🔄 הצרחה (Castling)
- ↩️ חזרה למצב קודם
- 📊 ספירת ניצחונות והפסדים
- ✨ אנימציות וממשק יפה

| רמה | תיאור |

| קל | מהלכים סבירים, בוחר מתוך 5 האפשרויות הטובות |
| בינוני | מהלכים טובים, לא תמיד הכי טובים |

1. בחר רמת קושי
2. לחץ על כלי כדי לבחור אותו
3. לחץ על משבצת היעד כדי לבצע מהלך
4. המחשב ישחק אחריך
